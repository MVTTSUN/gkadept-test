# Тестовое задание

## Использованные технологии

* Vite
* React
* TypeScript
* Redux Toolkit
* styled-components
* Faker
* FSD

## Деплой

[Vercel](https://gkadept-test.vercel.app/)

## Запуск проекта

Запуск в режиме разработчика:

```console
yarn dev
```

Cборка:

```console
yarn build
```

Запуск в режиме предварительного просмотра:

```console
yarn preview
```
